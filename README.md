# ChuGUI

ChuGUI is an immediate-mode 2D UI library for ChuGL.

## Components

- Icon
- Label
- Rect
- Meter
- Button - Momentary and Toggle
- Checkbox
- Color Picker
- Dropdown
- Input
- Knob
- Radio
- Slider - Continuous and Discrete
